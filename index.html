How the Web Works

The web functions thanks to communication of many computers. When I log onto www.udacity.com, my computer sends a HTTP Request to a server. HTTP stands for Hypertext Transfer Protocol. The server locates the corresponding HTML document and sends it back to my computer where my browser interprets the web page and displays it on my screen. This video explains this concept.

HTML

HTML stands for Hypertext Markup Language. HTML documents make up most of the content that is found on the web. HTML documents contain text conent which describes what someone sees and markup, which describes how something looks. This video covers this further.

Tags and Elements

HTML documents are made of HTML elements. When we compose an HTML document, we are telling the browser the type of each element by using HTML tags. This video covers this material.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them. Inline elements are placed on the same line as any text next to it.
